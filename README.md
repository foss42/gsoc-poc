# GSoC Proof of Concepts

Fork this repo & create a new branch.
Create a new folder inside `2026` with the name `yourname_project_name`.
Add your PoC inside it and send across a PR.

- PoCs that involve MCP testing must go thorough this [resource](https://dev.to/aws/how-i-built-mcp-apps-based-sales-analytics-agentic-ui-deployed-it-on-amazon-bedrock-agentcore-4e9i) and demonstrate the testing of the Sales Analytics MCP Apps server provided in the article. 
- AI Agents PoCs that include calling API Dash MCP server must go thorough this [resource](https://dev.to/aws/how-i-built-mcp-apps-based-sales-analytics-agentic-ui-deployed-it-on-amazon-bedrock-agentcore-4e9i) to understand the Sales Analytics MCP Apps server, then modify the source code provided in the article to create API Dash MCP Apps. 
- AI Eval PoCs must also explore go thorough this [resource](https://dev.to/aws/how-i-built-mcp-apps-based-sales-analytics-agentic-ui-deployed-it-on-amazon-bedrock-agentcore-4e9i) and explore if AI evaluation UI can be built using it to make it easy for end users to run evals from inside agent.
